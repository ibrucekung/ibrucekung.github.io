Android之TeamManager项目简介

本系统是基于GIS的团队管理Android应用，采用B/S架构进行Android客户端与服务器端开发，调用百度地图API提供地图服务。

Posted by ibrucekung on August 22, 2015

TeamManager

Version : 1.0

Development platform : Android Studio 1.3

Gradle Version : 2.5

Gradle Plugin Version : 1.3.0

简介

基于GIS的团队管理Android应用，本系统以Android为移动端平台，采用B/S架构进行Android客户端与服务器端开发，调用百度地图API进行地图开发，综合利用移动端定位技术和数据库技术、移动客户端与服务器端通信技术、WCF技术等实现了系统的各项功能。该系统主要实现的功能包括在实现传统团队管理的任务分发、日志记录、即时通讯的功能基础上，结合移动GIS技术实现对团队内部所有成员的准确实时追踪定位、轨迹回放、周边查询、紧急救援路线导航，同时团队管理员可以实现对内部成员的实时监控和轨迹回放等操作。

模块介绍

模块主要内容

  1. 登录界面
  2. 团队界面
  3. 通讯界面
  4. 个人信息界面
  5. 整个录屏内容

功能分析

登录界面

  • 包括头像，第一次登录时默认的是系统图标
  • 密码输入窗口
  • 忘记密码功能按钮
  • 注册功能按钮

地图界面

  • 显示当前位置信息
  • 切换显示的地图图层》普通地图 或 遥感图像
  • 是否记录运动轨迹
  • 放大和缩小
  • 查看缩放比例
  • 查找某地区的某位置

团队界面

  • 查找想要添加的团队
  • 创建团队
  • 管理员查看并管理团队信息，比如设置位置共享，解散团队（创建者）
  • 创建或查看公告
  • 创建或查看任务
  • 查看团队地图，可以查看到团队中启用位置共享的成员位置
  • 管理团队成员，比如：创建管理员
  • 团队聊天
  • 邀请好友加入当前团队
  • 管理员可以监控团队成员信息

通讯界面

  • 查找好友，查看或进行聊天
  • 聊天

个人信息

  • 查看或修改个人信息
  • 签到，签出
  • 我的任务
  • 我的日志
  • 我的日程计划
  • 我的轨迹
  • 我的警示点
  • 我的兴趣点
  • 我的兴趣点热力图

整个录屏内容

  • 包含了上面全部的录制过程
